## Installation
1. Clone repo:  
`git clone git@github.com:fabortwell/kittens_api.git`
2. Install dependencies:  
`bundle install`
3. Setup database:  
`rails db:create db:migrate db:seed`
4. Start server:  
`rails server`
5. Visit `http://localhost:3000`

## Seeding Sample Data
Pre-loaded with 5 adorable kittens:  
`rails db:seed`
